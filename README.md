# tap-duel-game

⚡ Tap Duel - Real-Time Multiplayer Web Game
Tap Duel is a real-time multiplayer reflex game where two players compete to tap the screen first after the "GO!" signal appears. The game uses Socket.io for instant communication, making gameplay smooth and synchronized across devices.

📌 Features
Real-time multiplayer gameplay
Player name entry
Automatic player matching
Countdown before each round
Random "GO!" signal
Reaction time display
Instant winner detection
Play Again option
Mobile-friendly interface
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Node.js
Express.js
Socket.io
Git & GitHub
Render
💡 How It Works
Player 1 enters a name and joins the game.
Player 2 joins using the same link.
A countdown starts automatically.
After a random delay, the screen displays GO!
The first player to tap wins the round.
Both players can restart and play again.
🎯 Challenges Faced
Synchronizing two players in real time
Preventing early taps
Detecting the first valid tap accurately
Managing player connections and disconnections
